# project-clean-energy-transition-tracker
***
This repository includes a data analysis project I did to accompanied dashboard elaboration.

Feel free to check the dashboard [here](https://public.tableau.com/app/profile/nur.adhyaksa.hamid/viz/TransitionProgressionTracker/Dashboard1).

## Description

This project starts with processing the data obtained from Ember-Climate web. The process will carried on to Tableau environment, to build a dashboard that fulfill the goals.

Below is a summary of comprehension table containing the actions I took to get this project to completion:

|Comprehension|Technique|Tool|
|:---|:---|:---|
|Data preparation|Clean, Modify|Python (pandas)|
|Data manipulation & exploration|Joins, Ranking, Pivot Table, Statistical Measurement|Python (pandas, NumPy, SciPy)|
|Data visualization|- |Tableau, Adobe Illustrator|
|Reporting|Markdown language|Visual Studio Code|
